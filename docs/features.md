# Features

## Clipboard History

ClipKit automatically captures text and images copied to the clipboard. Items are displayed in a searchable, scrollable list.

- **Text and images** — Both content types are captured automatically
- **Duplicate detection** — Copying the same content moves it to the top instead of creating a duplicate
- **One-click restore** — Click any item to copy it back to your clipboard

## Pinned Items

Save frequently used snippets as pinned items. Pinned items persist across app restarts and system reboots.

- **Configurable limit** — Store up to 50 pinned items (default: 12)
- **Drag & drop reordering** — Arrange pinned items in your preferred order
- **Persistent storage** — Pinned items are saved to disk and survive app updates

## Ephemeral Items

Recent clipboard history is stored as ephemeral items for the current session.

- **Session-based** — Automatically cleared on system reboot
- **Configurable limit** — Keep between 10 and 500 recent items (default: 50)

## Search & Filtering

Quickly find items in your clipboard history:

- **Real-time search** — Filter items as you type
- **Sort options** — Sort by most recent or alphabetically
- **Type filtering** — Show all items, text only, or images only

## Settings

Customize ClipKit's behavior:

| Setting | Range | Default |
|---------|-------|---------|
| Max pinned items | 1–50 | 12 |
| Max ephemeral items | 10–500 | 50 |
| Polling interval | Configurable | 0.5s |
| Launch at Login | On/Off | Off |
| Clear history on quit | On/Off | Off |
| Global shortcut | Configurable | Cmd+Shift+V |
