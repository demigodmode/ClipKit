# Keyboard Shortcuts

## Global Shortcut

| Shortcut | Action |
|----------|--------|
| **Cmd+Shift+V** | Show/Hide ClipKit |

The global shortcut works from any application. It can be customized in ClipKit's settings.

!!! note
    The global shortcut requires **Accessibility** permission. Grant this in **System Settings > Privacy & Security > Accessibility**. Enable **Launch at Login** for always-available access.

## In-App Shortcuts

| Shortcut | Action |
|----------|--------|
| **Up / Down** | Navigate items |
| **Return** | Copy selected item to clipboard |
| **Delete** | Remove selected item |
| **P** | Pin or unpin selected item |
| **Escape** | Clear selection |
